# Styling MDX Content

![Styling MDX]

**Author:** Vladislav Lovric  
**Published On:** 2024-10-16

MDX content can be styled effectively using various methods. In this post, we
will explore how to apply CSS and styled-components to your MDX content.

## 1. Using CSS Modules

You can create a CSS module for specific styling:

```css
/* styles.module.css */
.heading {
  color: #ff5600;
  font-size: 2rem;
}
```
